# EvolvingSpace

Gioco mobile 2D **idle/incremental spaziale** in pixel art, orientamento portrait (Unity 6). Questo repository ospita le **build Android** e la **documentazione** del progetto.

## 📱 Build Android

L'ultima APK è nelle [**Releases**](https://github.com/s-ricci/evolvingspace-builds/releases) — link diretto: [EvolvingSpace.apk](https://github.com/s-ricci/evolvingspace-builds/releases/latest/download/EvolvingSpace.apk).

Il gioco controlla da solo gli aggiornamenti all'avvio e propone il download della versione nuova; installandola sopra la vecchia i salvataggi si conservano.
